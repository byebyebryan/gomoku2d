# Lab Report Artifacts

Curated report data used by the web `/lab/` page.

These files are tracked product artifacts generated by the Rust lab:

- `bot-report.json` is copied into published builds as `/bot-report/report.json`.
- `analysis-report.json` is copied into published builds as `/analysis-report/report.json`.

Scratch and full diagnostic reports still belong in ignored
`gomoku-bot-lab/outputs/`.

## Refresh Bot Report

Use the canonical command in
[`Tournament Eval`](../../docs/reference/ops/tournament.md).

## Refresh Analysis Report

Run from `gomoku-bot-lab/` after the bot report is current. Use the canonical
command in [`Tournament Eval`](../../docs/reference/ops/tournament.md).

## Lab Comparisons

Use [`bot-report.json`](bot-report.json) as the current anchor source for
focused gauntlets and comparisons. See
[`Tournament Eval`](../../docs/reference/ops/tournament.md) for examples.
