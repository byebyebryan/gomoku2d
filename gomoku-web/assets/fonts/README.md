# Gomoku2D Fonts

PixelOperator font files used by the DOM shell and board sequence numbers.

Fonts are external runtime dependencies. The published `/assets/` route focuses
on project-authored sprites and icons instead of showcasing font variants.

## Runtime Fonts

- `PixelOperator.ttf`: body copy and general UI text
- `PixelOperator-Bold.ttf`: display text, labels, and buttons
- `PixelOperator8-Bold.ttf`: board result/replay sequence numbers

The other variants are kept as source options for future UI or canvas tuning.
